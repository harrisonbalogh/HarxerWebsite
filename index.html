      • HXDesknotes

        HXDesknotes pins editable text space to the Mac desktop. It's style matches that of the dock and other desktop interface features and has a minimal application footprint. This software makes taking a quick note easy and intuitive.

      • HXTheatre

        HXTheatre is a video media player. The software uses features like resuming from pause point when application is quit, autoplaying next sequential media, and skipping repeated intros for media of the same title. This software makes viewing chronological video files more in line with modern video players.
